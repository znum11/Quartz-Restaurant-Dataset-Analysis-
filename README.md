# Quartz-Restaurant-Dataset-Analysis-
This is an analysis conducted on data from a restaurant, 'Quartz' to analyse and understand customer dining patterns and spending behaviour based on demographics and preferences, to improve upon marketing strategies and personalise customer experiences using PowerBI.

Objective:- Analyse customer dining patterns and spending behaviour based on demographics and preferences, to improve marketing strategies and personalise customer experiences.
Methodologies:- Data Cleaning and Analysis by removing and handling empty cells, duplicated data and standardising the data to create better visuals.

•	Made a clear layout. 
•	Used slicers for interactive data exploration. 
•	Implemented KPIS and charts to highlight key metrics. 

Tools used: PowerBI 

Visualisations used:-
This dashboard uses a variety of visualisations to present key information about the Quartz Café & Grill. KPI cards are used at the top to display important metrics such as Loyalty Program Members, Average Total Rating, High Satisfaction Customers, and Unique Customers.
